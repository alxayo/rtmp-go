# Archived Documentation

This folder contains historical documentation that was generated during development
and debugging sessions. These files are preserved for reference but are not required
reading for understanding the current codebase.

## Contents

### Fix Notes (`fixes/`)
Debugging session notes from October-November 2025, documenting specific issues
and their resolutions:
- Connection timeout fixes
- OBS compatibility fixes  
- Malformed packet handling
- Relay sequence header issues
- Multi-destination relay error resolution

### Completed Feature Implementation Notes
Detailed implementation notes from features that are now fully integrated:
- Media logging implementation details
- Recording implementation details
- Relay implementation session notes

### Superseded Implementation Plans
Task breakdowns and implementation plans that were completed and are now
superseded by the actual code and its doc.go files.

## When to Read These

- If you're debugging a similar issue and want historical context
- If you want to understand why a specific design decision was made
- If you're curious about the project's development history
